<h1>Install</h1>
<b>Development</b>
<ol>
    <li>npm install</li>
    <li>npm start</li>
</ol>
<b>Production (at Heroku)</b>
<ol>
    <li>Push this git</li>
    <li>Push to Heroku</li>
    <li>Install Heroku addon: <br> 'heroku addon:create heroku-postresql:hobby-dev'</li>
    <li>Wait a bit for Heroku to complete the installation<br>'heroku pg:wait'</li>
    <li>Profit!</li>
</ol>
